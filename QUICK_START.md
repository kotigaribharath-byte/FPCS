## 🔧 Customization Ideas

### Change Colors
Edit these in `static/css/style.css`:
```css
:root {
    --primary: #0066ff;      /* Blue - change to your brand color */
    --secondary: #7c3aed;    /* Purple */
    --accent: #fbbf24;       /* Amber */
}
```

### Change Fonts
Look for the `font-family` rules in CSS

### Add Your Logo
Replace the text logo in `templates/base.html`:
- Line with `<i class="fas fa-chart-line"></i>` 
- Replace with `<img src="..." alt="logo">`

### Modify Theme
Change from dark to light theme by switching gradient colors
